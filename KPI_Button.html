Trend Graphs

1. Select the year you would like to review by clicking on the year bar (Current year by default).

2. Select time period (Weekly, Monthly, Quaterly) you would like to review by selecting on "Time Period"(Weekly by default).

3. Select the week/month/quarter you would like to review by clicking on the week/month/quarter bar.

4. If you want to check a specific site, select the site contributor you would like to review by clicking on the site contributors table.

5. If you want to check a specific reason, select the reason contributor you would like to review by clicking on the reason contributors table.

6. Visualization:

  • Year over Year - KPI: will show the KPI trend over the years for the selected Time Period and Site Structure.

  • KPI Diff - Year VS previous Year: will show the KPI difference between the selected Year and the Previous Year.

  • Weekly/Monthly/Quaterly KPI: will show the KPI trend over the weeks/months/quarters for the selected Year and Site Structure.

  • Weekly/Monthly/Quaterly KPI - Top 5 Reasons: will show the KPI trend over the weeks/months/quarters for the selected Year and Site Structure + Reason.

  • KPI Diff - Week/Month/Quarter VS Previous Week/Month/Quarter: will show the KPI difference between the selected Week/Month/Quarter and the Previous Week/Month/Quarter.

  • Weekly/Monthly/Quaterly - Site Contributors: will show the sites sorted by worst KPI difference between the selected Week/Month/Quarter and the Previous Week/Month/Quarter.

  • Weekly/Monthly/Quaterly - Top 5 Reasons: will show the Top 5 reasons sorted by worst KPI difference between the selected Week/Month/Quarter and the Previous Week/Month/Quarter.

  • Site Structure - Dive Deep Graphs will update for the selected Time Period.


    • Site Structure - Dive Deep

    1. Select the Tech Pillar/Region Pillar (drill down) you would like to review by clicking on the Tech Pillar/Region Pillar bar.

    2. Select the Technology you would like to review by clicking on Technology bar.

    3. Select the Site you would like to review by clicking on Site bar.

    4. Select the Area/Subarea (drill down) you would like to review by clicking on Area/Subarea bar.

    5. Select the MP you would like to review by clicking on MP table.

    6. Select the Reason you would like to review by clicking on Reason bar.

    7. Visualization:

  • KPI by Tech Pillar/Region Pillar: will show the KPI comparision between the different Tech Pillars/Region Pillars for the selected Time Period and upstream Site Structure.

  • KPI by Site: will show the KPI comparision between the different Sites for the selected Time Period and upstream Site Structure.

  • KPI by Area/Subarea: will show the KPI comparision between the different Areas/Subareas for the selected Time Period and upstream Site Structure.

  • TOP Worst - KPI MP's: will show the Top MP's sorted by worst KPI for the selected Time Period and upstream Site Structure.

  • TOP Worst - KPI MP's/Reason: will show the Top MP/Reason sorted by worst KPI for the selected Time Period and upstream Site Structure.

  • Trend Graphs will update for the selected Site Structure.


    This screen can be closed by clicking on the button again.
